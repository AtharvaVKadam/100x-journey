# Paytm Wallet Clone (MERN)

A full-stack wallet application built to demonstrate secure authentication and ACID database transactions.

## Tech Stack
* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Security:** JWT, Zod Validation

## Core Features
* Secure user signup and signin.
* Search for other users on the platform.
* Transfer money securely between accounts using MongoDB database transactions.
